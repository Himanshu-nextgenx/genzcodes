1xx – Informational (rarely used)
100 Continue → Request received, continue sending body
🟢 2xx – Success
200 OK → Request successful (most common)
201 Created → New resource created (POST)
202 Accepted → Request accepted, processing later
204 No Content → Success but no response body

🔵 3xx – Redirection
301 Moved Permanently → Permanent redirect
302 Found → Temporary redirect
304 Not Modified → Cached version use karo

🟡 4xx – Client Errors
400 Bad Request → Invalid request (missing fields, etc.)
401 Unauthorized → Login required
403 Forbidden → Access denied (login hai but permission nahi)
404 Not Found → Resource not found
405 Method Not Allowed → Wrong HTTP method
408 Request Timeout → Request timeout
409 Conflict → Duplicate / conflict (e.g. user already exists)
413 Payload Too Large → File size too big
415 Unsupported Media Type → Wrong content type
422 Unprocessable Entity → Validation error (very important in APIs)
429 Too Many Requests → Rate limit exceeded


🔴 5xx – Server Errors
500 Internal Server Error → Generic server error
501 Not Implemented → Feature not implemented
502 Bad Gateway → Upstream server error
503 Service Unavailable → Server down / maintenance
504 Gateway Timeout → Server took too long
